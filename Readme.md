## Bakait
------------------------
### An object oriented 2D game-engine in Clojure

#### Object Oriented?
Yes. Bakait is built on top of [CljOS](http://github.com/divs1210/cljos).

#### Try it out
Run `(-main 1)` in `bakait.core-test`.
